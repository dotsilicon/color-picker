# 🎨 Color Picker

**Catch the perfect dot**

Color Picker is a lightweight desktop utility for effortlessly grabbing the exact color from any pixel on your screen. Whether you're a designer, developer, or just color-curious, this app makes it simple to catch and copy colors instantly.

---

## 📦 Included in the Download

Download the `Color_Picker.zip` file and extract its contents:

```
Color_Picker.zip
│
├── ColorPicker.exe        # The main executable
├── README.md              # This documentation
└── Terms_of_Use.txt       # Legal and usage terms
```

---

## 🖥 How to Use

1. **Download & Extract** the `Color_Picker.zip`.
2. **Run** `ColorPicker.exe` (no installation required).
3. **Hover & Click** anywhere on the screen to pick a color.
4. The color is instantly copied to your clipboard in HEX format.
5. Press `Esc` or close the app to exit.

---

## ✅ Features

- Pixel-perfect color picking from anywhere on screen  
- Real-time magnified view  
- Instant HEX/RGB copy to clipboard  
- Portable — no installation or internet needed  
- Built for high-DPI displays
