# 🎨 Color Picker
**Catch the perfect dot**

A lightweight, fast, and precise desktop color picker designed for designers, developers, and anyone who needs accurate color values from anywhere on the screen.

---

## 🚀 Features

- 🎯 Screen Color Picker – Pick any color from anywhere on your screen  
- 🎨 Live Preview – See color changes in real-time while moving the cursor  
- 🌈 Color Formats:
  - HEX
  - RGB
  - CMYK  
- 🧩 Shade Generator – Automatically generate lighter and darker shades  
- 📋 One-Click Copy – Click any value to copy instantly  
- 🖱️ Custom Color Palette – Built-in visual picker with hue and saturation control  
- 🧠 Lightweight & Fast – No heavy background processes  
- 🖥️ Clean UI – Minimal and distraction-free

## 🛠️ Usage

1. Launch the application  
2. Click **"🎯 Select Color"**  
3. Move your cursor anywhere on the screen  
4. Click to capture the color  
5. Copy HEX, RGB, or CMYK values instantly
